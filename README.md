# 使用说明

> 需要 `node` 环境，并且全局安装了 `gulp`

## 涉及的插件
```js
browser-sync              自动刷新，静态服务
gulp-load-plugins         自动加载gulp插件
gulp-autoprefixer         设置浏览器版本自动处理浏览器前缀
gulp-csso                 压缩优化css
gulp-sass                 编译sass
gulp-sourcemaps           生层map文件
gulp-clean                删除文件
```

## 初次运行
```js
npm install 
```

## 启动
```js
gulp default 
```
